# 🌦 Weather Dashboard (React.js)

A modern, responsive weather application built using **React.js** and the Open-Meteo API.
It provides real-time weather data, 3-day forecasts, and hourly temperature charts with a clean UI.

---

## 🚀 Features

* 📍 **Auto Location Detection (GPS)**
* 🔍 **Search Weather by City**
* 🌡 **Current Weather Details**
* 📅 **3-Day Forecast**
* 📈 **Hourly Temperature Chart**
* 🌦 **Dynamic Weather Icons**
* 🖼 **Dynamic Background (Day/Night)**
* 📱 **Fully Responsive Design**
* ⚡ **Optimized Performance**

---

## 🛠 Tech Stack

* **Frontend:** React.js, JavaScript (ES6)
* **Styling:** CSS (Glassmorphism UI)
* **Charts:** Recharts
* **API:** Open-Meteo Weather API

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/Mr-Shahrukh/weather-dashboard.git
```

2. Navigate to project folder:

```
cd weather-dashboard
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```
